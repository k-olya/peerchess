## A game engine written in typescript

### Install

```
npm install
```

### Features:

- [x] **ECS** pattern implementation
- [x] cached **entity queries**
- [x] utility library: id generation, object lifecycle, events, math
- [x] peer.js-based multiplayer
- [x] **tailwind** for html ui
